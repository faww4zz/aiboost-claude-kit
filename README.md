# AI Boost Center ŌĆö Claude Code Starter Kit

A starter kit for new Claude Code users. Built for AI Boost Center training participants, but free for anyone.

**One install bundles six skills** ŌĆö they work straight away, no extra setup, no `npm`, no internet downloads after install:

- **`/prompt-master`** ŌĆö turns a rough idea into one clean, ready-to-paste prompt for any AI tool.
- **`/humanizer`** ŌĆö strips the AI tells out of any text so it reads like you wrote it.
- **`/researcher`** ŌĆö deep, multi-source research with real sources.
- **`/caveman`** ŌĆö same answer, far fewer words. Fun, and saves tokens.
- **`/find-skills`** ŌĆö finds and installs more skills, just describe what you want.
- **`/frontend-slides`** ŌĆö builds slide decks, dashboards, and small web pages.

Plus two commands:

- **`/welcome`** ŌĆö a plain-language orientation for your first hour on Claude Code.
- **`/setup-kit`** ŌĆö installs the one extra tool (superpowers) that ships separately.

---

## How to install

In Claude Code (desktop app, or the terminal), run these two lines:

```
/plugin marketplace add faww4zz/aiboost-claude-kit
/plugin install aiboost-starter
```

That's it. If a permission box pops up, click **Allow**. No admin rights needed ŌĆö it installs to your own user folder.

Then type `/welcome` to get started, and `/setup-kit` to add superpowers.

---

## Prefer one plain-English prompt? Paste this into Claude Code

If the slash commands feel fiddly, just paste this message to Claude and it will do it for you:

> Add the marketplace `faww4zz/aiboost-claude-kit` and install the `aiboost-starter` plugin from it. Then add the marketplace `anthropics/claude-plugins-official` and install the `superpowers` plugin. After each step, tell me whether it worked. If a permission prompt appears, tell me what to click.

---

## If the plugin install is blocked on your machine

Some work computers block plugin installs. If `/plugin install` does not work, use the **drop-in folder** instead ŌĆö you copy the skills straight into Claude Code by hand. No install command, no admin rights.

See the [`dropin/`](dropin/) folder and follow [`dropin/INSTALL.md`](dropin/INSTALL.md). In short: copy the `skills` and `commands` folders into your `~/.claude/` folder, restart Claude Code, done.
